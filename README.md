# Browns Hotels & Resorts - Technician Portal

A premium mobile application designed for the maintenance and technical teams of Browns Hotels & Resorts. This portal allows technicians to manage work orders, track schedules, and maintain elite hospitality standards across all properties.

## 🏨 Project Overview

This application has been custom-branded with the **Navy & Gold** identity of Browns Hotels. It provides a stable, high-performance interface for managing complex hotel maintenance tasks.

## ✨ Key Features

- **Elite Auth Flow**: Professional Welcome screen leading to secure Login and Registration pages.
- **Service Dashboard**: Real-time stats on pending and completed jobs, plus "Active Job" quick access.
- **My Schedule**: A comprehensive list of upcoming maintenance tasks with priority labeling.
- **Technician Profile**: Personalized profile with rating, experience tracking, and account security.
- **Modern UI**: Clean, high-contrast design built for efficiency and legibility.

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS)
- Expo Go app on your mobile device

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the project:
   ```bash
   npx expo start
   ```

3. Scan the QR code with your **Expo Go** app to view the portal.

## 🛠 Tech Stack

- **Framework**: Expo SDK 54 (React Native)
- **Navigation**: Expo Router (File-based)
- **Styling**: Standard React Native StyleSheet (Ensures 100% device compatibility and stability)
- **Icons**: Lucide React Native
- **Animations**: Optimized for smooth transitions
